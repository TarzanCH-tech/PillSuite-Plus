Frequently Asked Questions

How do automatic updates work?
PillSuite Plus checks for updates when the application starts (if enabled in settings) and periodically while running. When an update is found, you'll be notified and can choose to download and install it immediately or later. The update process preserves all your important files including your database, configurations, and custom themes.
What happens if an update fails?
The update system includes backup and restore functionality. If an update fails, the application will automatically restore from the backup created before the update process began, ensuring your system remains functional.
Can I manually update without using the built-in updater?
Yes, you can manually download update packages from this repository and install them. However, we recommend using the built-in updater as it handles file backups and proper installation automatically.
How do I report issues with an update?
If you encounter any issues with an update, please report them on our GitHub Issues page. Include details about the version you're updating from, the update version, and any error messages you received.
